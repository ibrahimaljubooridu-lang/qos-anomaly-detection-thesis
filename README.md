# Transformer-Based QoS Anomaly Detection

This repository contains the implementation of deep learning models for detecting Quality of Service (QoS) anomalies using the UNSW-NB15 dataset.

## Models
- CNN
- LSTM
- CNN-LSTM
- Transformer
- Autoencoder

## Methodology
- Feature selection
- Data preprocessing (scaling)
- Model training
- Evaluation (Accuracy, Precision, Recall, F1, ROC-AUC)

## Dataset
https://research.unsw.edu.au/projects/unsw-nb15-dataset

## How to Run
pip install -r requirements.txt

Open notebook and run all cells.
